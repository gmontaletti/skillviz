# ESCO ISCO group classification

Mapping table from ESCO level 4 occupation codes to profession names and
hierarchy labels.

## Usage

``` r
isco_gruppi
```

## Format

A data.table with columns:

- code:

  ESCO occupation code

- preferredLabel:

  Profession name (Italian)

- esco_v0101_hier_label_1:

  Hierarchy level 1

- esco_v0101_hier_label_2:

  Hierarchy level 2

- esco_v0101_hier_label_3:

  Hierarchy level 3
